二、秒杀接口（优化后）

a、RabbitMQ四种交换机模式

一、Direct模式：跟消息队列差不多，注册一个队列，往里面发或者取。

二、Topic模式: 注册了队列后，还要注册交换机，在把交换机绑定各个队列，发送时有三个参数（交换机名字，绑定通道名字，内容）

三、Fanout模式：注册队列后，注册广播交换机，然后绑定队列，发送时直接往交换机发，自动广播到所有绑定的队列。

四、Header模式：注册交换机时有个map匹配机制，在发送时，不能直接发送字符串，必须封装成Message，里面放字节数组和MessageProperties（类似map，里面放入header与之前做匹配），匹配成功才发送，相当于做了一个发送限制。
